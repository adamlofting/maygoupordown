maygoupordown
=============

A utility app that tracks a number over time

## Dependencies

* Node.js & npm
* Bower: `npm install -g bower`
* grunt-cli: `npm install -g grunt-cli`
* mocha: `npm install -g mocha`
* heroku toolbelt
* postgres database


## Installation

* fork this repo
* `git clone https://github.com/{{yourusername}}/maygoupordown.git`
* `cd maygoupordown`
* `npm install`
* `bower install`


## Running the app:

* `grunt`
* `foreman start`
